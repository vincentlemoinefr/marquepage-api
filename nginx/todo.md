Nginx can do these tasks :

- IP whitelist / blacklist
- Rate limiting
- Force https
- Wrong https protocol / cipher
- Require 2 Way SSL
- Request gateway timeout
- Load balancing
- Disable unwanted HTTP request : HEAD, CONNECT, OPTIONS, TRACE...
- Http Logging
- Wrong or missing headers
